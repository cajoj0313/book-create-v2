## 2. 修改内容

### 2.1 世界观生成 Prompt (`_build_world_setting_prompt()`)

**位置**: 第 134-169 行

**新增内容**:
- 信息密度控制：每句 ≤2 信息点，每段 ≤3 术语
- 句子结构控制：单句 ≤40 字，长短句交替
- 过渡衔接控制：因果关系必须有过渡词
- 术语处理规则：首次定义 ≤15 字，晋升路径用"→"标注
- 语言风格：现代白话文，统一风格
- 输出前自检：5 条强制检查规则

### 2.2 大纲生成 Prompt (`_build_outline_prompt()`)

**位置**: 第 400-429 行

**新增内容**:
- arc_summary 概述：≤150 字，起因→发展→结局结构
- 章节规划：key_events ≤3 事件，标题 ≤15 字有画面感
- 伏笔计划：hint ≤30 字有具体画面，间隔 ≤50 章
- 语言风格：现代白话文，统一风格
- 输出前自检：4 条强制检查规则

### 2.3 章节正文 Prompt (`_build_chapter_prompt()`)

**位置**: 第 713-749 行

**新增内容**:
- 正文叙事：段落 100-200 字，句子节奏短60%/中30%/长10%
- 对话描写：占比 ≤30%，单句 ≤50 字，配动作/神态描写
- 场景描写：首次 ≤100 字，已有 ≤50 字，战斗 ≤200 字
- 情感描写：铺垫、用动作表现而非直接描述、过渡说明
- 语言风格：第三人称，现代白话文，统一风格
- 输出前自检：5 条强制检查规则

## 3. 核心规则汇总

| 内容类型 | 控制规则 |
|---------|---------|
| **信息密度** | 每句 ≤2 信息点，每段 ≤3 术语 |
| **句子长度** | 单句 ≤40 字 |
| **过渡衔接** | 因果关系必须有过渡词 |
| **句子节奏** | 短句60% / 中句30% / 长句10% |
| **段落长度** | 100-200 字（正文） |
| **对话占比** | ≤30%（正文） |
| **arc_summary** | ≤150 字，每句1事件 |
| **章节标题** | ≤15 字，有画面感 |
| **伏笔提示** | ≤30 字，有具体画面 |
