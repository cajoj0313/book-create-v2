## 3. 完成内容

### 3.1 后端变更

**文件**: `backend/src/application/generation_service.py`

| 方法 | 变更 |
|------|------|
| `_build_random_world_setting_prompt(genre)` | **新增** - 构建都市职场随机生成 Prompt |
| `stream_generate_world_setting()` | **修改** - 新增 `random_generate` 和 `genre` 参数 |

**Prompt 特点**:
- 主题选择：职场晋升/商业博弈/创业奋斗/职场爱情
- 背景设定：现代都市、城市地理、职场层级
- 能力体系：职场等级（实习生→高管）
- 核心冲突：晋升竞争/商业博弈/职场阴谋
- 复用写作风格指南：信息密度/句子结构/过渡衔接

**文件**: `backend/src/interfaces/api_generation.py`

| 变更 | 内容 |
|------|------|
| `GenerateWorldSettingRequest` | 新增 `random_generate: bool = False` |
| `GenerateWorldSettingRequest` | 新增 `genre: str = "都市职场"` |
| `user_description` | 改为可选（随机生成时可忽略） |

---

### 3.2 前端变更

**文件**: `web-front/src/pages/WorldBuilder.tsx`

| 变更 | 内容 |
|------|------|
| `handleRandomGenerate()` | **新增** - 随机生成交互函数 |
| "AI 随机生成"按钮 | **新增** - indigo 色系，与"开始生成"并列 |

**交互逻辑**:
- 无需检查描述输入是否为空
- SSE 请求参数: `{ random_generate: true, genre: "都市职场" }`
- 复用现有 SSE 客户端处理流式响应

**按钮布局**:
```
[开始生成] (vermilion)  [AI 随机生成] (indigo)
```

## 7. 用户验证建议

### 功能验证步骤

1. **启动服务**
   ```bash
   cd backend && ./start.sh
   cd web-front && pnpm dev
   ```

2. **访问世界观构建页面**

   - 确认流程不受影响
