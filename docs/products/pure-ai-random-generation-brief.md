## 1. 需求概述

### 核心诉求 ⭐

用户明确强调：**不要有预生成的配置数据，所有的小说内容，均需要从大模型中获取。**

### 当前问题分析

**问题定位**: 当前 Prompt 中包含大量"预配置选项列表"，限制了 AI 的创意自由度。

| 问题类型 | 具体位置 | 预配置内容 | 影响 |
|----------|----------|------------|------|
| 男主类型固定 | `_build_random_world_setting_prompt()` 第 271-275 行 | 霸道总裁/暖男医生/腹黑律师/创业精英（4 种固定类型） | AI 只能从 4 种中选择，无法创新 |
| 女主类型固定 | 同上第 277-280 行 | 职场新人/小透明/白富美（3 种固定类型） | AI 只能从 3 种中选择，无法创新 |
| 感情线类型固定 | 同上第 282-286 行 | 误会型/情敌型/职场型/命运型（4 种固定类型） | 情节模式被限定 |
| 城市选择固定 | 同上第 291-293 行 | 上海/北京/深圳/杭州/广州（5 个固定城市） | 场景缺乏新意 |
| 工作场所固定 | 同上第 294 行 | 大型企业/创业公司/律师事务所/医院/设计公司（5 种固定） | 背景单一 |

**示例代码片段**（generation_service.py 第 271-280 行）：
```python
###男主类型选择（随机选择其一）
- 霸道总裁：强势、富有、护短、冷峻外表下有深情
- 暖男医生：温和、细心、体贴、治愈系
- 腹黑律师：聪明、深沉、有城府、表面温和内心强势
- 创业精英：野心、魄力、年轻有为、白手起家

### 女主类型选择（随机选择其一）
- 职场新人：刚入职、努力、单纯、有成长空间
- 小透明：低调、善良、有潜力、默默付出
- 白富美：优雅、独立、有主见、家境优越
```

### 问题本质

| 维度 | 当前设计 | 用户期望 |
|------|----------|----------|
| **数据来源** | Prompt 内预配置选项 → AI 选择 | AI 完全自由创意生成 |
| **创意自由度** | 限定范围（4 种男主/3 种女主） | 无限定，AI 自由发挥 |
| **生成多样性** | 受限于预配置选项数量 | 无限可能 |
| **产品定位** | "从预设模板中选择生成" | "AI 纯随机创意生成" |

## 2. 用户故事

### 主故事

```
作为灵笔产品用户，
我希望所有小说内容（人物、背景、感情线等）完全由 AI 自由创意生成，
而不是从预配置的选项列表中选择，
以便获得真正随机、充满创意的多样化内容。
```

### 子故事

| 角色 | 需求 | 价值 |
|------|------|------|
| 用户 | AI 自由生成男主设定（不限类型） | 可能出现新颖的男主类型（如"电竞主播"、"美食博主"） |
| 用户 | AI 自由生成女主设定（不限类型） | 可能出现新颖的女主类型（如"独立创业者"、"海外留学归来"） |
| 用户 | AI 自由生成感情线模式 | 可能出现新颖的感情线（如"异地恋"、"跨国恋"） |
| 用户 | AI 自由生成城市和背景 | 可能出现新颖的背景（如"成都"、"重庆"、"新媒体公司"） |

### 保留内容（不应移除）

| 内容 | 保留原因 |
|------|----------|
| JSON 输出格式定义 | 结构化数据，便于解析和存储 |
| 写作风格指南（信息密度、句子长度等） | 质量控制，防止生成劣质内容 |
| 示例对比（好示例 vs 坏示例） | 质量学习，提升生成质量 |
| 约束条件（感情线完整性、结局圆满等） | 内容一致性保障 |

## 7. 技术方案

### Prompt 改造示例

**改造前**（预配置列表）：
```python
###男主类型选择（随机选择其一）
- 霸道总裁：强势、富有、护短、冷峻外表下有深情
- 暖男医生：温和、细心、体贴、治愈系
- 腹黑律师：聪明、深沉、有城府、表面温和内心强势
- 创业精英：野心、魄力、年轻有为、白手起家
```

**改造后**（自由生成）：
```python
### male_lead（男主设定）⭐ 自由创意生成
请自由创意生成男主设定，不限类型，要求：
- name: 现代风格姓名（2-3字）
- identity: 自由定义的身份类型（如霸道总裁、电竞主播、美食博主、健身教练等，不限预设类型）
- age: 28-35岁
- appearance: 外貌描述（≤30字，简洁具体）
- personality: 性格特点数组（3-5个，用具体词汇）
- wealth: 财富背景
- occupation: 具体职业

创意要求：
- 男主必须有魅力、有能力、有感情变化
- 可以是任何职业/身份，不限预设类型
- 性格要有独特性，不要千篇一律
```

## 9. 风险识别

| 风险类型 | 雎险描述 | 缓解措施 |
|---------|---------|---------|
| **AI 生成不稳定** | 移除限制后 AI 可能生成不合理的内容 | 保留写作风格指南和约束条件 |
| **JSON 格式不稳定** | AI 可能不按格式输出 | 保留 JSON 输出格式定义 + 严格解析 |
| **感情线不完整** | AI 可能跳过某些感情阶段 | 保留感情节奏框架约束 |
| **类型选择下拉框遗留** | 前端 FEAT-15 功能需同步处理 | 可保留为可选功能，或移除 |
